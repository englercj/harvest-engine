# `exceptions` node

Sets the exception handling model support generated by the compiler.

## Arguments

1. (string) - Required. The behavior of exception handling.
    * `default` - Use the toolset's default behavior. This is the default value.
    * `on` - Enable exception handling.
    * `off` - Disable exception handling.
    * `seh` - Enable exception handling, and structured exception handling.

## Properties

None.

## Children

None.

## Scopes

- [`module`](module_node.md)
- [`project`](project_node.md)

## Example

```kdl
exceptions off
```
